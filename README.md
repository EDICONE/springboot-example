## 项目

Spring Boot 系列学习示例.



## 开发环境

- Spring Boot 2.0.4
- MySql 8.0.12
- JDK 1.8
- IDEA 2018.2.1

- Spring Data JPA  => 查看 springboot-data-jpa
- MySQL 8 => 查看 springboot-data-jpa/springboot-mybatis
